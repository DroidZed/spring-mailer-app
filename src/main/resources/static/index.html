Spring Mailer App

This app is used as a backend for sending emails.

It's built using GraphQL and serving two functionalities.

sendEmail endpoint, used for..you guessed it, delivering emails üòã 
            
mutation deliverEmail {

	sendMail (

		mail: {

			from: "example@example.com"

			subject: "testing the api"
			body: "lorem ipsum"
		}
	) 
  {
		msg
	}
}
  • The mutation keyword is used to indicate that the client wants to mutate the data.
  • We call the mutation sendMail using the required parameters.
  • In retrun we expect a message which is an object containing the msg field.
